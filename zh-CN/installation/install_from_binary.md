# 二进制安装

我们目前只提供 64 位操作系统的二进制。

## 如何通过二进制升级？

1. 下载最新版的二进制 ZIP 压缩包。
2. 删除当前的 `templates` 目录。
3. 解压压缩包并将所有内容复制粘贴到相应位置。
